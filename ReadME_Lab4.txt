Lab 4 My Profile using Flask

Ashanti Louise Villadiego's Website

To run the application:
    1. install python if you haven't install yet
    1.1 go to your working dir 
    2. install flask
        pip install flask
    3. execute the command 
        python app.py 

This project contains:
    1. app.py - the main Python file containing the flask application
    2. templates - this folder contains the 7 HTML files used for different pages of the website
        *Area of a circle
        *Area of a triangle
        *Contacts
        *Index
        *Profile
        *Works
        *To Upper case
    3. static - contains the images used for the website
